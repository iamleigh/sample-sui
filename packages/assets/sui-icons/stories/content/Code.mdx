<Section title="Props" border={true} style={{ paddingTop: 0 }}>
	When utilizing the icon in the form of an SVG component, there are several properties available for customization.

	<PropSection title="title" type="string">
		Sets the display name for the generated icon component
    </PropSection>

	<PropSection title="colorScheme" type="string">
		The color scheme to apply to the icon, it accepts one of the following values:

		<List>
			<span label="neutral" />
			<span label="informative" />
			<span label="success" />
			<span label="warning" />
			<span label="critical" />
		</List>
    </PropSection>

	<PropSection title="fill" type="string">
		Custom color for the icon
    </PropSection>

	<PropSection title="size" type="string">
		The size of the icon, it accepts one of the following values:

		<List>
			<span label="xs" />
			<span label="sm" />
			<span label="md" />
			<span label="lg" />
			<span label="xl" />
		</List>
    </PropSection>

	<PropSection title="viewBox" type="string" defaultValue="0 0 24 24">
		Specify the aspect ratio and coordinate system of the SVG.
		<a href="https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/viewBox" target="_blank">learn more</a>
    </PropSection>

	<PropSection title="iconHeight" type="number" defaultValue="24">
		Specifies the height of the icon in pixels
    </PropSection>

	<PropSection title="iconWidth" type="number" defaultValue="24">
		Specifies the width of the icon in pixels
    </PropSection>

	<PropSection title="onClick" type="function">
		A callback function to be executed when the icon is clicked
    </PropSection>

	<PropSection title="className" type="string">
		Additional custom className for further styling
    </PropSection>
</Section>
